Blockquote

Blockquotes are defined with the <blockquote> tag. They are used to show longer quotations or excerpts from other text. Note: The following example shows a blockquote using the default green institutional color. This color will be determined by the station in which the content appears.

I have a theory that you can make any sentence seem profound by writing the name of a dead philosopher at the end of it.

Plato
Single Paragraph

<blockquote>...</blockquote>

Multiple Paragraphs

<blockquote>
  <p>...</p>
  <p>...</p>
</blockquote>

With Citation

<blockquote>
  <p>I have a theory...</p>
  <footer>Plato</footer>
</blockquote>

Bold

The <b> tag is used to set a portion of text off stylistically. It is used for keywords that are important, but not necessarily read aloud differently.

Turn left at the corner.

Corrections

The <del> tag is used in conjunction with the <ins> tag to denote corrections in articles after they have been published.

It was confirmed that Keith Moon John Bonham was the better drummer.

<del>...</del><ins>...</ins>

Strong Emphasis

The <strong> tag is used to provide strong emphasis to text. It is used for text that would be pronounced with urgency.

Warning: I-25 is closed until further notice!

<strong>...</strong>

Emphasis

The <em> tag is used to provide emphasis to text. It is used for text that would be pronounced differently.

We had to do something about it!

<em>...</em>

Headings

Heading tags <h1> - <h6> are used to title new sections of content. Note: <h1> is used for the title of an article and is automatically populated by the CMS. It is advised that you only use <h2> and below when authoring content.

Introduction

The intro class can be added to a <p> tag to create a larger body of text suitable if article summaries and introductions.

Lorem ipsum dolor sit amet, consectetur adipisicing elit, Ut enim ad minim veniam.

<p class="intro">...</p>

Italic

The <i> tag is used to italicize text, such as movie titles, as well as to denote words that should be read in an alternate voice such as transliterated foreign words.

It was a great Cinco de Mayo celebration!

<i>...</i>

Link

Links provide users a way to navigate from one page to another and are created with the <a> tag.

This is a inline link.

<a href="http://...">...</a>
