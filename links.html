List - Ruled

Ruled lists can be created by applying a list--ruled class to <ul> or <ol> tags and the list-itemclass to each <li> within.


<ul class="list--ruled">
  <li class="list-item">Unordered list Style: Ut enim ad minim veniam.</li>
  <li class="list-item">Quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.</li>
  <li class="list-item">Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.</li>
</ul>

List - Ordered

The <ol> tag is used to create a numbered list.


<ol>
  <li>Ordered list Style: Ut enim ad minim veniam.</li>
  <li>Quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.
    <ol>
      <li>Nested list Style.</li>
      <li>Quis nostrud exercitation ullamco laboris.</li>
    </ol>
  </li>
  <li>Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.</li>
</ol>


```
<ol>
  <li>...</li>
  <li>...</li>
  <li>...
    <ol>
      <li>...</li>
      <li>...</li>
    </ol>
</ol>
```
